## Azafure LLC (Calamity Mod):

**License:** [Calamity Mod License](https://github.com/CalamityTeam/CalamityModPublic/blob/1.4.4/LICENSE.md)

The following code in Arcadia has been reused from Calamity in its following files and folders:
- Their primitive system (in `Core/Graphics/Primitives`).
- Their particle system (in `Core/Graphics/Particles`).
- Their draw layer system (in `Core/Graphics/DrawLayers`).
- Their pixelation system (in `Core/Graphics/Pixelation`).
- Their window title system (in `Content/UI/WindowTitles.cs`).
- Their shader loading system (in `Assets/Effects/ArcadiaShaders.cs`).
- Some of their utilities (in `Core/Utilities`).
- Some of their global item tooltip code (in `Core/Globals/Items/ArcadiaGlobalItem.Tooltips.cs`).
- Some of their global NPC and NPC attribute code (in `Core/Globals/NPCs`).
